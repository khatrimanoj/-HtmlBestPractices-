# `body` etiketi ekleyin

Bazen `body` etiketi beklenmedik bir pozisyonda bir tarayıcı tarafından tamamlanmaktadır.

Yanlış:

    <html>
      <head>
        ...
      </head>
      ...
    </html>

Doğru:

    <html>
      <head>
        ...
      </head>
      <body>
        ...
      </body>
    </html>
